## Tabs in pure CSS
 
- First clone in your local repository project
```
git clone https://github.com/mateusocana/css-tabs.git
```
- To enter into the project repository
```
cd css-tabs
```
- to edit the HTML
```
vim index.html 
```
- to edit the CSS
```
vim style.css
```
- Press ESC and ```wp:``` to return on terminal
